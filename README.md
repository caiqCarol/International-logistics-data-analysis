# International-logistics-data-analysis

## 1. 业务问题
本项目聚焦国际物流公司的核心经营问题：

公司利润是否存在结构性风险？
不同运输方式（海运 / 空运 / 卡航）对利润的影响机制是什么？
是否存在成本结构不合理导致的利润损失？

目标：找出利润波动与成本结构的核心驱动因素，并提出优化策略

## 2. Dataset
基于国际物流业务逻辑，使用 Python 构建 10,000 条模拟订单数据，字段包括：

订单维度：order_id, order_date, customer_type
运输维度：transport_mode, origin_city, destination_country
成本维度：shipping_cost, customs_cost, warehouse_cost, total_cost
收入与利润：revenue, profit, profit_margin
履约指标：promised_days, actual_days, delay_flag

## 3. 方法
Python：数据清洗（缺失值 / 类型转换）
SQL：分组聚合、窗口函数、指标计算
分析维度：
成本结构拆解
不同运输方式利润对比
履约与利润关系分析

## 4. 核心发现
###### 成本结构呈现“运输方式驱动差异”
海运：运输成本占比 >90%，利润高度依赖运价
空运：运输成本占比下降，清关+仓储成本显著上升
卡航/铁路：成本结构更均衡，但附加成本占比高于海运
结论：
不同运输方式的成本驱动逻辑完全不同

###### 利润稳定，但存在结构性风险
各月利润整体稳定，无明显波动
但利润高度依赖运输成本控制（尤其海运）
风险点：
一旦运价上涨，利润可能被快速压缩

##### 高延误率 ≠ 短期利润影响
延误率长期维持在约 55%
对当前利润影响较小

说明：
当前利润模型未计入客户流失/赔偿等长期影响

## 5. 业务洞察

##### 1：海运是“利润放大器 + 风险源”
优势：低成本 → 高利润空间
风险：对运价极度敏感
本质：公司利润依赖外部市场（运价）

##### 2：空运是“效率型成本模型”
利润不取决于运输成本
而取决于：仓储 + 清关 + 操作效率
本质：这是“运营能力竞争”，不是价格竞争

##### 3：当前公司忽视“服务质量的长期价值”
延误率高但利润未受影响
说明：没有计入客户流失
这是一个隐藏风险

## 6. 建议

###### 成本优化策略
海运：加强运价谈判 / 优化航线选择
空运：优化清关流程 + 降低仓储周转时间

##### 运输结构优化
减少低利润空运订单占比
提高高利润运输方式占比

##### 履约优化（长期）
降低延误率（目标 <30%）
建立延误成本模型（客户流失 / 赔偿）
